## 📌 Project Description

This project implements three classical sorting algorithms and analyzes their performance using **empirical runtime analysis**.

The goal is to compare how different algorithms perform with varying input sizes by measuring their execution time.

---

## ⚙️ Algorithms Implemented

### 🔹 Insertion Sort

A simple sorting algorithm that builds the final sorted array one element at a time.

### 🔹 Merge Sort

A divide-and-conquer algorithm that splits the array into smaller parts and merges them in sorted order.

### 🔹 Quick Sort

An efficient sorting algorithm that uses a pivot element to partition the array into subarrays.
